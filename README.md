# ** Files change monitor **
        --by Phantom



* #### main.sh


> Usage: ```main.sh <path> <time>```


> path: path where to look for changes
time: how long back to look. eg. -2 seconds, -2 minutes, -7 days

>Sample: main.sh /root "-2 days"


* #### cdiff.sh

> Usage: ```cdiff.sh <path>```

>path: path where to look for changes

>Sample: cdiff.sh /root
